Tag HTML <data>

Exemplo

O exemplo a seguir exibe nomes de produtos e associa cada nome a um número de produto:

<ul>
  <li><data value="21053">Tomate</data></li>
  <li><data value="21054">Alface</data></li>
  <li><data value="21055">Batata</data></li>
</ul>

Definição e uso

A tag <data> é utilizada para adicionar uma tradução legível por máquina de um determinado conteúdo.

Este elemento fornece um valor legível por máquina para processadores de dados e um valor legível por humanos para renderização em um navegador.

Dica: Se o conteúdo estiver relacionado a hora ou data, use o elemento <time>.

Atributos

Atributo Valor Descrição
value machine-readable format Especifica a tradução legível por máquina do conteúdo do elemento
